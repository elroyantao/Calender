# Calender
A calender App



### Getting started

Download the source code
```sh
$ git clone https://github.com/elroyantao/maze.git
```

Goto the directory
```sh
$ cd Calender
```

Install dependency
```sh
$ npm install
```

Run
```sh
$ npm start
```



open link to view the app(If not automatically opened)
```sh
http://localhost:8080/
